# web_1lab_shop
# Web Shop — лабораторная №1

Интернет-магазин на чистых HTML/CSS/JavaScript: каталог товаров, корзина, оформление заказа.  
Фреймворки (Bootstrap/React/Vue/Angular) не использовались.

## Функциональность

- Каталог товаров: название, цена, изображение, кнопка «Добавить в корзину»
- Корзина:
  - добавление товаров
  - удаление товаров
  - изменение количества
  - автоматический пересчёт итоговой суммы
  - сохранение состояния в `localStorage` (корзина не пропадает после обновления страницы)
- Оформление заказа:
  - форма (имя, фамилия, адрес, телефон)
  - сообщение «Заказ создан!» после отправки

## Адаптивность

Корректное отображение на:
- ноутбуках
- планшетах
- смартфонах (адаптивная сетка через media queries)

##  Технологии

- HTML5 (семантическая разметка)
- CSS3 (Grid/Flex + media queries)

- localStorage

## Структура проекта

- `index.html` — разметка
- `css/style.css` — стили
- `js/` — логика (каталог/корзина/заказ)
- `assets/img/` — изображения товаров
